Definisaun Website

Website mak konjuntu pájina web (web pages) ne’ebé bele hare iha Internet, uza navegador hanesan Chrome, Firefox, ka Safari. Website sira halo ho HTML, CSS, no duni JavaScript (tanba buat sira ne’e halo estrutura, dezenhu, no funsionalidade).

Ita bele dehan:

Website mak lugar ida iha Internet ne’ebé fornese informasaun ka servisu ba utilizadór sira. Website bele iha página ida de'it (single page), ka barak (multi-page), no bele atu interativu ka simple.

HTML

HTML mak lian padraun atu halo estrutura ba website sira. HTML bele hatudu titulu, paragrafu, imajen, video, tabela, link, no buat seluk iha página web.

CSS

CSS (Cascading Style Sheets) mak linguajen estilu ne’ebé uza atu dezenha no bele bele forma bonita ba HTML iha website.

JAVA SCRIPTS

JavaScript (JS) mak linguajen programação ne’ebé uza iha web atu halo website bele interativu no dinamiku.
